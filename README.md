# HTML Resume Portfolio

This repository contains my **resume created using pure HTML**.  
The purpose of this project is to showcase my profile, skills, and experience in a **simple, clean, and accessible web format**.

---

## 📄 About the Project

- Resume built using **only HTML** (no frameworks)
- Clean and minimal layout
- Easy to view in any web browser
- Hosted using **GitHub Pages**

---

## 🚀 Live Demo

You can view the resume here:  
👉 https://akashguptadev.github.io/html-resume/

---

## 🛠️ Technologies Used

- HTML5

---

## 📌 Features

- Structured resume sections (Summary, Skills, Experience, Certifications)
- Center‑aligned header for better readability
- Simple footer with copyright information
- Lightweight and fast loading

## 🔗 Icons Used

The project uses **Unicode / Emoji icons**, which do not require any external libraries.

Examples:
- 📞 Phone
- ✉️ Email
- 📍 Location
- 🔗 LinkedIn / Links

These icons are supported natively by browsers.

You can explore more Unicode icons here:
- https://unicode-table.com/
